Build Instructions:
1. Drop all the files in the current directory into the FLIP server.
2. Open a bash terminal session and navigate to that directory.
3. Type "make" into the terminal to build the solution.
4. An executable called "tsp" should be built.
5. You can run the TSP algorithm by typing: tsp <inputfilename>
6. The output will be in the file: inputfilename.tour
7. To clean all the output files and delete the compiled exe, type: "make clean" into the console.
